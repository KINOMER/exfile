# exfile
定时将mysql中的数据导入到excel文件，后台运行
-----------------------------------  
1、python版本
-----------------------------------  
我是在2.7.5下编写的代码，测试通过
2、需要安装的库文件
-----------------------------------  
pip install xlwt
-----------------------------------  
pip install MySQL-python

3、注意修改数据库ip，用户名和密码哦
----------------------------------- 
4、注意修改保存文件的路径哦
----------------------------------- 
5、执行：python exfile.py
-----------------------------------
